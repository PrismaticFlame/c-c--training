I don't really have that much experience in these languages other than a few classes at my undergrad, so I think it's time that I beef myself up in this department. I had Claude create a lesson plan of sorts, but there will be NO AI used in the writing of any of the code in this project. Problems, bugs, un-runnable code? I will deal with it. What's the point otherwise?

The lesson plan is the html file in root. Check it out if you like. It is tailored to my experience, so likely won't work for everyone sadly.
